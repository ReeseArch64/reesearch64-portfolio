# **App Name**: AI Hub Portfolio

## Core Features:

- About Me Section: Dedicated 'About Me' section detailing the developer's 10-year journey and areas of expertise in AI, n8n automations, and web systems.
- Project Gallery: A visual gallery showcasing key projects, with filters for AI, n8n, and web systems.
- Technical Skills: Interactive list of technical skills, emphasizing proficiency in AI, n8n, and web development technologies.
- Contact Form: Simple contact form for inquiries.
- AI project spotlight: An AI-powered 'tool' which analyzes all project descriptions to find common topics, and uses those insights to group together, tag, and highlight groups of relevant projects for potential clients.

## Style Guidelines:

- Background color: Dark gray (#121212) for a modern dark mode theme.
- Primary color: Electric Blue (#7DF9FF) for highlighting interactive elements and key information.
- Accent color: Deep Blue (#003049) to complement and balance the electric blue.
- Body font: 'Inter', a grotesque-style sans-serif for clean and readable text.
- Headline font: 'Space Grotesk', a sans-serif, used for a techy and scientific feel. 
- Use a set of minimalist, line-based icons that align with the electric blue accent color to ensure UI consistency.
- Incorporate subtle transitions and animations for a professional touch.
